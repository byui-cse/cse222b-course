CSE 222a: Week 04

Ponder and Prove

Your Task

Before beginning this Task:

  • Make sure you have read and understood the reading for the week.

Situation

You are working for a company that needs a technology solution to track the containers of pharmaceuticals they have in stock available for sale to pharmacies that are their customers. A UML diagram has been provided, see Figure 1 below.

Task 1

Using classes and or structs, create a code set that appropriately uses lists, sets, and dictionaries to solve the problem. Also write driver code to execute the code you write. Attempt to add and remove many packages of many types of medication. Ensure, using stepwise debugging and debugging statements to validate that your code is behaving as the company intends.

The unique identifiers for the dictionary in the design are to follow the NDC database's package code pattern, [5 digits]-[3 digits]-[2 digits]. You may use real codes from the database along with real information, or you may make them up.

To follow good stock roll-over procedures, ensure that the containers sold are always the ones that have the nearest expiration data. The result of calling the currentStockOf function must also be sorted in the same order.

Review

After completing the previous steps:

  1. For each of the tasks, review one of the many possible solutions,
  2. Compare and contrast your code with the possible solution's code, and
  3. Make note of any improvements you could make for future tasks.
