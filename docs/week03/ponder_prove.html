Week 3 Task Instructions

Before beginning this Assignment:

  • Make sure you have read and understood the reading for the week.

Like last week, please download the tasks.kt and main.kt files for this week's assignment and create a Kotlin project that contains these two files. You should not modify main.kt in any way. Your tasks are in comments in tasks.kt. Each time you successfully compile and run, the test functions in main.kt will test your progress and report which tasks are successfully completed, which are currently failing on all of the tests.

Your goal is to write code in tasks.kt so that all tasks are passing.

The first task this week includes implementing the data structures in this UML diagram.

This is similar to, but not identical to the UML diagram in the reading this week. Since the test code refers to these classes and struct, the project will not initially compile until you have implemented the necessary data structures.

When all tasks are passing, submit your work in the Canvas system. You should submit two files: your copy of tasks.kt and a screenshot showing successful execution.
