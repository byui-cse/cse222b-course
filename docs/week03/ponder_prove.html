Your Task

Before beginning this Task:

  • Make sure you have read and understood the reading for the week.

Situation

You are working for a company that needs a technology solution to determine how many of what types of programmers they need to hire. They plan on dramatically increasing the technology staff and want to have 52 development leads when the expansion is complete. They have data in a database that contains all pertinent information regarding all current employees for the company. The company also has a series of rules that apply to programmers and their development leads. They are;

  1. The type of managers being tracked in the data set for this problem is Development Lead.
  2. All managers must manage at least 3 employees.
  3. Managers can manage no more than 15 employees.
  4. All programmer teammates must be managed by the same development lead.
  5. Each programmer with less than 5 years experience must have at least one teammate that has worked for the company for more than 1 year and has at least 5 years experience.
  6. No programmer can be a member of more than one team.

The company needs to know how quickly they can get to the point where they have 123 total programmers assuming all current programmers are retained. To this end, they need to know how many programmers they need to groom to become experienced so the expansion can happen. The data your code produces will be used to develop a hiring schedule.

Task 1

Using classes, create a code set that indicates what types of programmers need to be hired to meet the criteria above. Use classes and apply only IS_A inheritance between the classes you create. You will not have access to the employee database, so to test your application the Quality Assurance group has asked that you generate a testing data set to run your code against. This data set is to consist of 52 development leads and between 120 and 125 programmers. You are instructed to create the data set in such a way that the experience constraint is violated. This class_generate_data.swiftclass_generate_data file has been provided for you to meet this requirement. Use it as if you had written it as per the requirement. The generateData function is all you will need to call.

This UML diagram has been provided by the engineers at your company. Write code to match this UML diagram. The generateData function will use your implementation of these classes.

As part of the code set required by your employers,

  • create a function that assigns programmers to development leads based on the criteria stated, and
  • create a function using the result of the assignment function. Name this function generateNeededHires. It has a list of Managers as one of it's parameters and must return data describing the types of programmers that need to be hired.

Task 2

Recreate your solution for Task 1 using classes but this time use only HAS_A type inheritance.

Task 3

Recreate your solution for Task 1 using structures and protocols. This time choose between using either or both IS_A and HAS_A inheritance. Use your best judgment to make these decisions. You will need to judiciously modify the generate_data.swift file to make this work.

Review

After completing the previous steps:

  1. For each of the tasks, review one of the many possible solutions,
  2. Compare and contrast your code with the possible solution's code, and
  3. Make note of any improvements you could make for future tasks.
