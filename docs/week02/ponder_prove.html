Week 2 Task Instructions

Before beginning this Assignment:

  1. Make sure you have reviewed the Installation and Purpose and The Fundamentals course content.

Please download the tasks.kt and main.kt files for this week's assignment and create a Kotlin project that contains these two files. You should not modify main.ki in any way. You should be able to successfully compile the two files. When the project runs it will throw an error at the first broken test.

Your tasks are in comments in tasks.kt. Follow the instructions for each task and implement code to make it work. Each time you successfully compile and run, the test functions in main.kt will test your progress. When there are no failing tests, you have finished that tasks for this week.

Your goal is to write code in tasks.kt so that all tasks are passing.

When all tasks are passing, submit your work in the Canvas system. You should submit two files: your copy of tasks.kt and a screenshot showing successful execution.
