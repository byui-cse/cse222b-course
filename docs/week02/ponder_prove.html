Week 2 Task Instructions

The Fundamentals

Before beginning this Task:

  1. Be part of a team consisting of 2-3 members.
  2. Make sure you have reviewed the Installation and Purpose and The Fundamentals course content.

Task 1

Create a variable named diastolicBloodPressures and initialize it to be an Array of Arrays of Integers. There are to be 100 sub-Arrays in the diastolicBloodPressurs Array, each representing diastolic blood pressures taken and recorded over some period of time. Each of these sub-Arrays is to contain 40 random Integers between 60 and 130. Put this code in the main function in the main.kt file.

The diastolicBloodPressures Array is the parameter for a function called generateArrayStats that is to be put in a file named stats_lib.kt. This function will return an Array of Arrays of Doubles. Each sub-Array (row) of the result being returned is to have, in this order, the mean, maximum, and minimum of the corresponding sub-Array in the data.

Create another function called repeatedlyPrintRequestedStats that has as a parameter an Array of Arrays of Doubles. Put it in a file called patient_ui_lib.kt. It should safely and repeatedly print out each requested patient's information. The user will enter a 'patient number' in the terminal that you are to interpret as being the index of the sub-array to print. You are also to warn the user if they enter an invalid patient number. Feel free to use part of the sample code from last week as a template, beginning point, for this function.

Stretch Challenge

Add median and mode to the statistics for each row.

Review

After completing the previous steps:

  1. Review one of the many possible solutions for this task,
  2. Compare and contrast your code with the possible solution's code, and
  3. Make note of any improvements you could make for future tasks.
